# QazaqTest — сайт компании по продаже лабораторного оборудования

Статический сайт (HTML/CSS/JS, без сборки и зависимостей) для компании из Казахстана,
которая поставляет лабораторное оборудование, мебель, посуду и реактивы.

## Структура

```
index.html            — главная: hero, категории, преимущества, популярные товары, этапы работы, CTA
catalog.html          — каталог с поиском и фильтрами по категориям (?cat=...)
about.html            — о компании
contacts.html         — контакты, форма обратной связи, карта
assets/css/style.css  — все стили (дизайн-система в CSS-переменных в :root)
assets/js/products.js — данные каталога (категории, товары, иконки)
assets/js/main.js     — шапка, мобильное меню, модалка заявки, формы, анимации
assets/js/catalog.js  — фильтрация и поиск на странице каталога
assets/img/           — фавикон и графика
```

## Как запустить локально

Сборка не нужна — достаточно любого статического сервера:

```bash
python3 -m http.server 8000
# открыть http://localhost:8000
```

Либо просто открыть `index.html` в браузере.

## Как публиковать

Подходит любой статический хостинг: GitHub Pages, Netlify, Vercel, Cloudflare Pages.
Для GitHub Pages: Settings → Pages → Deploy from branch → выбрать ветку и корень `/`.
